# [대소문자 바꿔서 출력하기 - 181949](https://velog.io/@bluepaper14_/%EB%8C%80%EC%86%8C%EB%AC%B8%EC%9E%90%EB%B0%94%EA%BF%94%EC%84%9C%EC%B6%9C%EB%A0%A5%ED%95%98%EA%B8%B0-181949)

![](https://velog.velcdn.com/images/bluepaper14_/post/13d44a24-f22c-410e-b134-16a9a326c375/image.png)

### 문제 설명

영어 알파벳으로 이루어진 문자열 `str`이 주어집니다. 각 알파벳을 대문자는 소문자로 소문자는 대문자로 변환해서 출력하는 코드를 작성해
보세요.

* * *

##### 제한사항

  * 1 ≤ `str`의 길이 ≤ 20 
    * `str`은 알파벳으로 이루어진 문자열입니다.

* * *

##### 입출력 예

입력 #1

    
    
    aBcDeFg
    

출력 #1

    
    
    AbCdEfG
    

* * *

### 문제정의

이번 문제는 문자열이 주어졌을때 이를 대문자에서 소문자로 소문자를 대문자로 만드는 방법을 숙지하면된다. 이는 아스키 코드 이해에 있으며
함수를 사용해서도 가독성 있게 만들어보자.

### 초기코드

    
    
    #include <iostream>
    #include <string>
    
    using namespace std;
    
    int main(void) {
        string str;
        cin >> str;
        return 0;
    }

초기코드이다. 이를 for문으로 문자열에 대한 한음절씩 꺼내 대문자 소문자 여부를 확인하고 이를 변환하도록 만들어봤다.

    
    
    #include <iostream>
    #include <string>
    
    using namespace std;
    
    int main(void) {
        string str;
        cin >> str;
    
        for(int i = 0; i < str.length(); i++) {
            if(str[i] >= 'a' && str[i] <= 'z')
                str[i] = str[i] - 32;
            else if (str[i] >= 'A' && str[i] <= 'Z') {
                str[i] = str[i] + 32;
            }
        }
        cout << str;
        return 0;
    }

예를 들어 'A'라는 아스키코드로(실제 내부값) 은 65이다. 'a'는 97이다. 즉 _대문자와 소문자는 같은 선상의 숫자 32가
차이난다._

> A B C ........ a b c

이렇게 말이다. 이제 분기를 소문자인지 대문자인지 따지려면

    
    
    if(str[i] >= 'a' && str[i] <= 'z)

즉 해당 음절이 소문자 a보다 크고 z보다 작으면 소문자이니 이를 32를 빼주어 대문자로 변환하였다.

반대로 함수를 사용해서 이를 재구성해보자.

    
    
    #include <iostream>
    #include <string>
    #include <cctype>   // toupper, tolower
    
    using namespace std;
    
    int main() {
        string str;
        cin >> str;
    
        for (int i = 0; i < str.length(); i++) {
            if (islower(str[i])) {
                str[i] = toupper(str[i]);
            } else if (isupper(str[i])) {
                str[i] = tolower(str[i]);
            }
        }
    
        cout << str;
        return 0;
    }

islower() 와 isupper()은 대문자와 소문자인지 판별하는 함수이다.

또한 toupper() 그리고 tolower()은 각각 대문자 소문자를 변환하는함수이다. 이는 cctype 헤더를 사용해서 만들 수 있다.

### 기억해야하는점

> 문자는 아스키 코드로 정리됨. 그리고 string은 배열처럼 인덱스로 접근가능하고 헤더파일 사용시 함수를 사용하여 가독성 높일 수 있다.
