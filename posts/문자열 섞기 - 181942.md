# [문자열 섞기 - 181942](https://velog.io/@bluepaper14_/%EB%AC%B8%EC%9E%90%EC%97%B4%EC%84%9E%EA%B8%B0-181942)

![](https://velog.velcdn.com/images/bluepaper14_/post/e568b20f-ea8f-486a-8998-543cdbc35109/image.png)

### 문제 설명

길이가 같은 두 문자열 `str1`과 `str2`가 주어집니다.

두 문자열의 각 문자가 앞에서부터 서로 번갈아가면서 한 번씩 등장하는 문자열을 만들어 return 하는 solution 함수를 완성해
주세요.

* * *

##### 제한사항

  * 1 ≤ `str1`의 길이 = `str2`의 길이 ≤ 10 
    * `str1`과 `str2`는 알파벳 소문자로 이루어진 문자열입니다.

* * *

##### 입출력 예

str1 | str2 | result  
---|---|---  
"aaaaa" | "bbbbb" | "ababababab"  
  
### 문제정의

이번 문제는 answer에 각 문자열을 번갈아 넣으면 된다. 여러가지 방법들이 있다.

### 초기코드

    
    
    #include <string>
    #include <vector>
    
    using namespace std;
    
    string solution(string str1, string str2) {
        string answer = "";
        return answer;
    }

문제를 풀때 str1 str2 각각 배열의 위치를 뽑아보면 쉽다. 예를 들어 최종출력을 보면 ababababab인데

str1 은 0 2 4 6 8 에 삽입되어 있고 str2 는 1 3 5 7 9 이다. 즉 홀수 짝수로 나눠서 삽입하면된다. 하지만 반복문을
통해 문자열을 삽입하려면 answer에 대한 문자열 크기를 주어져야한다. 그래서 resize라는 함수를 사용한다.

    
    
    #include <string>
    #include <vector>
    
    using namespace std;
    
    string solution(string str1, string str2) {
        string answer;
        answer.resize(str1.length() + str2.length());
    
        for (int i = 0; i < str1.length(); i++) {
            answer[2*i] = str1[i];
            answer[2*i + 1] = str2[i];
        }
    
        return answer;
    }
    

위를 코드를 보면 resize를 통해 answer의 크기를 지정해줬고 이후 반복문을 통해 answer 배열에 각 str1 str2의 문자열을
삽입해줬다. 01 23 45 67 이런식으로 말이다.

또 다른 방법이 있다. 굳이 answer에 대한 크기 지정없이 삽입한다.

    
    
    #include <string>
    #include <vector>
    
    using namespace std;
    
    string solution(string str1, string str2) {
        string answer = "";
    
        for(int i=0; i<str1.length(); i++)
        {
            answer.push_back(str1[i]);
            answer.push_back(str2[i]);
        }
        return answer;
    }

push_back을 사용해서 각 삽입하는 문자열을 answer에 쉽게 넣었다.
